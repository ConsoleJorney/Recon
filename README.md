# Recon

Jogo shooter espacial arcade desenvolvido em JavaScript puro, com movimentação livre em ambiente 2D.

##  Funcionalidades
- Movimento livre em todas as direções
- Sistema de disparo
- Detecção de colisão
- Atualização em tempo real

## Tecnologias
- JavaScript (Vanilla)
- HTML5 Canvas

##  Objetivo
Explorar conceitos de desenvolvimento de jogos 2D, incluindo física básica, renderização e interação em tempo real.
